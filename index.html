How to Get a Gemini API Key

  1. Visit Google AI Studio at aistudio.google.com/app/apikey
  2. Sign in with your Google account
  3. Click on "Get API key" button
  4. Choose "Create API key in new project" or select an existing project
  5. Copy the generated API key
  6. Paste it in the API Key field above

Note: The API key will only be stored in your browser session and never sent to any server except Google's Gemini API.
